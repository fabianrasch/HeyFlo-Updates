## What is HeyFlo?

HeyFlo is a native macOS dictation app. Press **Fn**, speak, release — your words appear in whatever app is in focus. No copy-paste. No switching windows. Just talk.

Built for people who switch between German and English throughout the day. HeyFlo handles both natively, cleans up your speech with AI, and formats the output for the context you're writing in — email, Slack, Notion, code comments.

## Features

- **Press Fn to dictate** — works in every macOS app, no integration needed
- **German + English + Denglisch** — switch mid-sentence, HeyFlo follows
- **AI cleanup** — removes filler words, fixes punctuation, formats for context
- **< 1 second latency** — fast enough to feel like typing
- **Liquid Glass design** — feels native to macOS Sequoia
- **Privacy-first** — no recordings stored, no training on your data
- **Auto-updates** — always on the latest version via Sparkle

## Installation

1. Download `HeyFlo.dmg` from [Releases](https://github.com/fabianrasch/HeyFlo-Updates/releases/latest)
2. Open the DMG — drag **HeyFlo** into **Applications**
3. Launch HeyFlo and grant microphone permission when prompted
4. Press **Fn** anywhere and start speaking

## Auto-Updates

HeyFlo updates itself automatically via [Sparkle](https://sparkle-project.org/). When a new version ships, you'll see a notification inside the app. No manual re-download needed.

This repository hosts the release artifacts and the Sparkle appcast feed. The app checks this feed periodically to detect new versions.
